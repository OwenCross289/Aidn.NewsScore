### News Score

Repository for Aidn technical assesment focusing on calculating the NEWS Score of a patient based on vital signs.

## Assumptions 
- The request cannot be changed, so the request has to come in via a collection of vital signs.
- There can only be 3 vital signs per request.
- No duplicate vital signs
- No missing vital signs
- The request and response has to be in JSON format, assuming the example is taken from JS where object literals are used.
